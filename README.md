### Start Project

install project dependencies

```shell
$ npm i
```

start in development environment

```shell
$ npm run start
```

---

### Package as an installer

build as html static files

```shell
$ npm run build
```

package a windows 64-bit installer

```shell
$ npm run pak-win-x64
```

package a windows 32-bit installer

```shell
$ npm run pak-win-ia32
```
